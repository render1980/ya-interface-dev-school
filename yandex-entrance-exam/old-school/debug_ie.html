Использование инструментов отладки в Internet Explorer

Инструментарий, входящий в состав Developer Tools, который интегрирован в IE, начиная с версии 8, во многом схож с инструментарием плагина Firebug. Те же вкладки CSS, HTML, Script (Сценарий), Net (Сеть). Нет, разве что средства для просмотра структуры DOM. Зато есть вкладка Profile (Профили), как и в Chrome. Она пригодится в целях оптимизации загрузки страницы.

Сценарий отладки схож с другими браузерами: использование средств вкладки Сценарий и Консоль.

Отдельно стоит упомянуть вкладку Профили. При помощи кнопки "Запуск создания профилей/Остановка создания профилей", задав режим (Текущее представление), можно просмотреть объекты, которые инициируются при загрузке страницы и время, затраченное на них.

Отталкиваясь от полученных данных, можно оптимизировать скрипт для увеличения производительности работы страницы.

Другие версии IE рассматривать не буду в расчете на актуальность версий данного браузера у большинства пользователей.
